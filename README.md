# [Floating Featured Image](https://development.azurecurve.co.uk/classicpress-plugins/floating-featured-image/)
![Plugin Banner](/assets/pluginimages/banner-1544x500.png)

# Description

[featured-image] shortcode allows a floating featured image to be placed at the top of a post or page.

Featured Images are added as a custom post type; iamges can be linked to a category, tag or using a custom link.

This plugin is multisite compatible; each site will need settings to be configured in the admin dashboard.

# Installation Instructions

 * Download the latest release of the plugin from [GitHub](https://github.com/azurecurve/azrcrv-floating-featured-image/releases/latest/).
 * Upload the entire zip file using the Plugins upload function in your ClassicPress admin panel.
 * Activate the plugin.
 * Configure relevant settings via the configuration page in the admin control panel (azurecurve menu).
